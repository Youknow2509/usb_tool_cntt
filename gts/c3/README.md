- Phương pháp tiếp tuyến của đường cong `y = f(x)` tại điểm `M(x, f(x))` có dạng `y = f'(x0)(x - x0) + f(x0)`.

Ta tính được: `x1 = x0 - f(x0)/f'(x0)`
